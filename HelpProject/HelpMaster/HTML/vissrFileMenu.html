File Menu

Sequence, annotation, Patman data is loaded into VisSR using
  • Open sequence file...
  • Open annotations file...
  • Open Patman file...

The View PAREsnip results... menu item prompts the user for an output file from PAREsnip and then displays the data in tiers corresponding to category system used by PAREsnip.

An image of the main data view is created by clicking the Take picture... item.
