The Tools

The UEA sRNA Workbench is a suite of tools used for the analysis of sRNA datasets.

Each separate tool in the sRNA Workbench is accessed through the Tools Menu. A user can select from here the desired tool for the type of analysis required.

Additionally certain tools such as miRCat allow users to chain certain tools together and load certain tools directly from sets of results. Please see the related documentation for further details.
