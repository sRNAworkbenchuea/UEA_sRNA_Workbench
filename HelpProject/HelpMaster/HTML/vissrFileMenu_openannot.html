Open Annotations File

The 'Open annotations file...' menu-item lets the user choose an annotations file in GFF format. The user is then prompted for the annotation types which are to be loaded. The arrow buttons between the two lists copy all of the items in the direction indicated. To copy a single item double-click on its name.


If possible, the types 'gene', 'miRNA', 'pseudogene' and 'transposable_element' are selected by default. When 'OK' is clicked, a tier is created for each selected value.
