View results

All loci predicted in SiLoCo can be visualised in VisSR using the "Show in VisSR" menu item located in the View menu (note for the majority of datasets this is likely to be a slow operation). Alternatively, each locus can be sent individually to VisSR by bringing up a context menu on a particular locus and selecting the "Show in VisSR" button (note this is a much faster way of viewing individual loci).
