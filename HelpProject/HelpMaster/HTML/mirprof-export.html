Exporting results

miRProf can export two types of file: a results table in .csv format and a list of sRNAs matching known miRNAs (in FastA format). The results table contains a formatted list of reads that match to known miRNAs. It also contains information about redundant (total) and non-redundant (distinct) sequence counts in the input set before and after every filtering step. The csv file can be loaded into any good spreadsheet program. The miRNAs in FastA format maybe used for to identify and extract miRNAs from existing sRNA datasets.
