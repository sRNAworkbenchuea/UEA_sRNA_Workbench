UEA sRNA Workbench Help System

Activating help is achieved through the Help menus at the top of the main sRNA Workbench window and for each tool.

Activating the help system for a specific tool will put the help into context sensitive mode and open a targeted page helping a user find their way to the desired information.

Activating the help from the main sRNA Workbench window will bring a user directly to the index of the help pages
